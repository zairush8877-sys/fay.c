# Fay Concept — анализ рынка и стратегия сайта

*Июль 2026 · подготовлено для fay.concept (Нальчик, ул. Лермонтова, 4)*

## 1. Кто такие Fay Concept

Команда начинала как байеры, сейчас это **concept store + консьерж-сервис**:
«тихая curated-зона», где каждая вещь проходит отбор, а поиск и подбор идут
через личного консьержа. Каналы: Instagram (10,5 тыс. подписчиков), WhatsApp,
Telegram-канал, офлайн-пространство в Нальчике.

## 2. Состояние рынка (РФ, 2025–2026)

- После ухода международных брендов рынок стал многополярным: ниши занимают
  российские марки, турецкие сети и азиатские игроки; конкуренция ужесточилась,
  доля убыточных сетей одежды выросла с 18% до 34%.
- **Параллельный импорт** сделал ассортимент менее предсказуемым, а цены выше
  (логистика, маржа посредников, валюта) — именно здесь ценность байера,
  который гарантирует оригинал и адекватную цену.
- **Ресейл люкса растёт**: рынок б/у люксовых вещей достиг ~253 млрд ₽ (+10% за
  год); ресейл-корнеры открывают ЦУМ и Oskelly — доверие к «проверенным вещам»
  становится валютой.
- Премиальный e-commerce конкурирует с маркетплейсами не ценой, а
  **неценовыми стимулами**: подборки стилистов, закрытые форматы продаж,
  атмосфера и сервис. Это ровно позиционирование Fay.

## 3. Конкурентное поле

| Сегмент | Игроки | Сильные стороны | Слабые стороны (шанс для Fay) |
|---|---|---|---|
| Массовые байер-платформы | CDEK Shopping, USmall, Бандеролька, LiteMF, ShopoTam | Масштаб, логистика, низкие комиссии | Нет кураторства и стиля, «каталог всего», обезличенный сервис |
| Люкс-ритейл | ЦУМ, КМ20 | Ассортимент, бренд, офлайн-опыт | Москва-центричность, высокие цены, дистанция с клиентом |
| Ресейл | Oskelly, ресейл-корнеры | Цена, редкие позиции | Б/у, вопросы состояния и подлинности |
| Instagram-байеры | Тысячи частных байеров | Гибкость, личный контакт | Нет гарантий, нет витрины, нет системности |

**Ниша Fay Concept:** курируемый люкс + личный сервис + офлайн-точка в регионе,
где федеральные игроки почти не присутствуют. Модель «AI отбирает — человек
решает» (как у Stitch Fix: клиент получает не тысячи позиций, а персональную
подборку из 5–10) — это фактически формализация того, что Fay уже делает
вручную.

## 4. Тренды технологий, которые стоит использовать

- **ИИ-подбор и виртуальная примерка** — главный технологический тренд
  fashion e-commerce 2025–2026: возвраты на маркетплейсах достигают 30–40%,
  виртуальная примерка снижает их на ~25%.
- **Гиперперсонализация**: анализ предпочтений → короткая подборка вместо
  каталога. Идеально ложится на консьерж-модель.
- Квиз-подбор на сайте — это ещё и **сбор данных о клиенте** (стиль, бюджет,
  размеры) до первого диалога с консьержем: заявка приходит уже «тёплой».

## 5. Что делает сайт (этот репозиторий)

1. **Витрина доверия** — премиальный лендинг в эстетике бренда: кто мы,
   как работает консьерж-ритуал, привилегии, адрес и контакты.
2. **ИИ-стилист Fay** — интерактивный подбор образа: пол, повод, стиль,
   бюджет, палитра → алгоритм ранжирует каталог и собирает персональную
   подборку; её можно одной кнопкой отправить консьержу в WhatsApp.
   Работает полностью на клиенте (без бэкенда), при желании подключается
   Claude API для генеративных рекомендаций.
3. **Каталог-витрина** с фильтрами по категориям и кнопкой «запросить у
   консьержа» на каждой позиции — лидогенерация в WhatsApp/Telegram.

## 6. Дорожная карта после запуска

- Заменить плейсхолдеры каталога реальными фото и позициями (данные — один
  JSON-массив в `script.js`).
- Подключить примерку по фото (интеграция с сервисом виртуальной примерки)
  и/или Claude API для «живого» диалога со стилистом.
- Telegram-бот с теми же рекомендациями; CRM-учёт заявок из квиза.
- Домен + GitHub Pages/hosting, метрика (Яндекс.Метрика), микроразметка
  для локального поиска «Нальчик».
