# Fay Concept — сайт магазина и консьерж-сервиса

Одностраничный сайт для [fay.concept](https://instagram.com/fay.concept) —
concept store и консьерж-сервиса в Нальчике (ул. Лермонтова, 4).

## Что внутри

- **Лендинг** в эстетике бренда: кто мы, ритуал заказа из 6 шагов, привилегии, контакты.
- **ИИ-стилист Fay** — квиз из 5 вопросов (для кого, повод, стиль, бюджет, палитра):
  алгоритм ранжирует витрину и собирает персональный образ, который одной кнопкой
  отправляется консьержу в WhatsApp с готовым текстом заявки.
- **Витрина** с фильтрами по категориям и кнопкой «Запросить» на каждой позиции.
- Анализ рынка и стратегия — в [MARKET_ANALYSIS.md](MARKET_ANALYSIS.md).

Статичный HTML/CSS/JS без сборки и без бэкенда — открывается локально
(`index.html`) и публикуется на GitHub Pages в один клик
(Settings → Pages → Deploy from branch).

## Как наполнять витрину

Все позиции — массив `PRODUCTS` в начале `script.js`. Для реальных фото
положите файлы в папку `img/` и добавьте позиции поле `image: "img/file.jpg"` —
карточка автоматически покажет фото вместо цветной заливки. Поля `styles`,
`occasions`, `palette`, `gender`, `price` использует ИИ-стилист для подбора.

Контакты (WhatsApp, Instagram, taplink) заданы в `index.html` и константой
`WHATSAPP` в `script.js`.

## Апгрейд до «живого» ИИ

Локальный движок рекомендаций — функция `recommend()` в `script.js`.
Чтобы стилист вёл диалог и объяснял выбор, замените её вызовом Claude API
(модель `claude-sonnet-5` подойдёт по цене/качеству): передавайте анкету
и массив `PRODUCTS` в prompt и просите вернуть JSON с id выбранных позиций
и комментарием стилиста. Понадобится лёгкий серверный прокси для ключа API.
